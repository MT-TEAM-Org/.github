# PlayHive

스포츠 팬들이 모이는 **욕설 없는 클린 커뮤니티**  
E스포츠 · 축구 · 야구 세 가지 카테고리를 중심으로  
뉴스, 중계, 커뮤니티 기능을 제공합니다.

<br />

## Demo

- **Live URL:** 준비중

<br />

## 서비스 개요

**PlayHive**는 E스포츠, 축구, 야구 팬들이  
욕설 없이 소통할 수 있도록 설계된 **클린 스포츠 커뮤니티**입니다.  
각 카테고리별 게시판과 뉴스 크롤링, E스포츠 경기 중계 기능을 제공합니다.

<br />

## 주요 기능

| 구분                | 설명                                                |
| ------------------- | --------------------------------------------------- |
| **카테고리 게시판** | E스포츠 / 축구 / 야구 별 독립 게시판                |
| **뉴스 피드**       | 카테고리별 최신 기사 자동 크롤링                    |
| **E스포츠 중계**    | YouTube 임베드로 실시간 경기 중계 제공              |
| **욕설 필터링**     | 라이브러리(`bad-words`) 기반 욕설 자동 감지 및 치환 |
| **댓글/좋아요**     | 게시글 반응 및 커뮤니티 참여 기능                   |
| **검색/정렬**       | 종목·인기순·최신순 탐색                             |
| **신고/제재**       | 부적절한 게시글/댓글 신고 기능                      |
| **마이페이지**      | 내 활동 내역, 작성글/댓글 관리                      |
| **반응형 UI**       | PC / Tablet / Mobile 완벽 대응                      |
